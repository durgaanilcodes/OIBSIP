# Random Password Generator

## 📌 Description
A Python application that generates secure random passwords based on user preferences.

## 🚀 Features
- User chooses password length
- Supports uppercase letters
- Supports lowercase letters
- Supports numbers
- Supports symbols
- Input validation
- Option to generate multiple passwords

## 🛠 Technologies Used
- Python 3.11
- VS Code

## ▶️ How to Run

```bash
python password_generator.py
```
